# App Privacy Checklist (App Store Connect)

Use this as a working draft while filling App Privacy.

## Data Types

### Financial Info
- Type: Financial Information (User-entered budget/expense data)
- Collected by app: Yes (user enters it)
- Linked to user identity: No (no account system in v1.1)
- Used for tracking: No
- Purpose: App functionality

### Contact Info
- Collected by app: No

### Location
- Collected by app: No

### Browsing/Search History
- Collected by app: No

### Identifiers
- Collected by app: No third-party identifier collection in v1.1

### Diagnostics
- Collected by app: No external diagnostics collection by default

## Tracking
- Does this app track users across apps and websites owned by other companies?
- Suggested answer: No

## Sensitive Data
- Health data: No
- Contacts: No
- Photos: No
- Microphone/Camera: No

## Notes
If you later add analytics SDKs, ad SDKs, or account-based cloud backends, update these answers before new release submission.
